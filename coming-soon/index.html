Features

Alles was du brauchst

🏢

Multi-Mandant

Beliebig viele Firmen mit eigenen Vorlagen, Logos und SMB-Einstellungen verwalten.

📝

Template-Editor

HTML-, TXT- und RTF-Vorlagen mit Code-Editor, Live-Vorschau und Variablen-Referenz.

📄

CSV-Import

Mitarbeiter per CSV importieren. Bestehende Datensätze werden automatisch aktualisiert.

⚡

Auto-Generierung

HTM, TXT und RTF Signaturen für alle Mitarbeiter auf Knopfdruck generieren.

🚀

SMB-Deploy

Signaturen direkt auf einen Netzwerk-Share hochladen. Alte Dateien werden bereinigt.

🌐

Zweisprachig

Komplette Oberfläche in Deutsch und Englisch. Umschaltbar per Klick.

So einfach geht's

In 3 Schritten zur Signatur

1

Firma & Vorlage

Mandant anlegen und Signatur-Vorlage im Editor gestalten. Live-Vorschau inklusive.

2

Mitarbeiter importieren

CSV hochladen oder manuell anlegen. Vorhandene Mitarbeiter werden per E-Mail erkannt.

3

Generieren & Deployen

Signaturen generieren und per SMB direkt auf das Netzlaufwerk hochladen. Fertig.

Self-Hosted

Deine Daten bleiben bei dir

Ein Docker-Container, eine SQLite-Datenbank. Kein Cloud-Abo, keine Abhängigkeiten. Läuft auf jedem Server, NAS oder Homelab.

🐳 Docker 🐍 Python / Flask 🗃 SQLite 🔒 MIT License
docker-compose.yml
# docker compose up -d services: signature-tool: image: ghcr.io/onlinecrash24/email-signature-manager:latest container_name: email-signature-manager network_mode: host environment: - SECRET_KEY=change-me-secret-key - ADMIN_USER=admin - ADMIN_PASSWORD=MySecurePassword123 - LISTEN_PORT=5010 - DEFAULT_LANG=de volumes: - signature-data:/opt/signature-tool/data restart: unless-stopped volumes: signature-data:
